# 🐶 Dogppuccin — A Canine‑Inspired Theme Pack for Visual Studio Code

Dogppuccin is a warm, harmonious, dog‑themed reinterpretation of the popular Catppuccin palette.  
Four variants, four breeds, one cozy coding experience.

## 🎨 Variants

- **Latte (Husky)** — Light, clean, crisp  
- **Frappe (Beagle)** — Soft, warm, balanced  
- **Macchiato (Golden Retriever)** — Smooth, golden, comforting  
- **Mocha (German Shepherd)** — Dark, bold, focused  

Each theme is handcrafted with harmonious palettes and subtle contrasts for long coding sessions.

---

## 📦 Local Installation (without Marketplace)

Dogppuccin can be installed locally using a `.vsix` package.

### **1. Download the file**

Go to the **Releases** section of this repository and download:

- `dogppuccin-1.0.0.vsix`

If the version number changes, simply download the latest `.vsix` available.

### **2. Install in Visual Studio Code**

1. Open VS Code  
2. Go to **Extensions**  
3. Click the three dots `⋯` in the top‑right corner  
4. Select **Install from VSIX…**  
5. Choose the `.vsix` file  

The theme will appear under **Preferences → Color Theme**.
