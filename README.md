# 🐶 Dogppuccin

Dogppuccin is a collection of Visual Studio Code themes **inspired by Catppuccin**, reimagined with a canine identity.  
All palettes and themes were **AI‑generated** and refined for visual harmony, AA/AAA contrast, and cross‑variant consistency.

> ⚠️ Dogppuccin is an independent project, not affiliated with or endorsed by the Catppuccin team.  
> All palettes and themes were generated with AI.

---

## 🎨 Variants

### 🐾 Latte — Husky (cool, clean)
A cold, elegant palette with icy tones and subtle blue accents.

### 🐾 Frappe — Beagle (neutral, balanced)
Warm neutrals, beige and brown tones with friendly highlights.

### 🐾 Macchiato — Golden Retriever (warm, soft)
A cozy, creamy palette inspired by golden fur and warm sunlight.

### 🐾 Mocha — German Shepherd (dark, bold)
Deep, professional dark tones with amber accents.
